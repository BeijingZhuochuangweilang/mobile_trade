# 液厂用户登陆系统

1. 微信关注公众号《掌易助理》

2. 进入公众号后，点击“进入平台”

3. 输入“公司名称”“姓名”，点击“绑定”

4. 允许小程序获取并验证手机号，如非常用手机号，请点击下方“使用其他号码“

5. 点击获取并验证手机号后进入公司主页

# 司机用户登录系统

1. 微信关注公众号《掌易助理》

2. 进入公众号后，点击“承运信息”

3. 输入“姓名”“身份证号”，点击“绑定”

4. 允许小程序获取并验证手机号，如非常用手机号，请点击下方“使用其他号码“

5. 点击获取并验证手机号后进入承运信息页面，可显示当日承运信息
